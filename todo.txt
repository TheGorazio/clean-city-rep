Чистый город 

1. Клиент 
React + Redux SPA приложение
    Модель урны:
        заполненность
        батарея
        маркер на карте 
            координаты
            адрес
        прогнозирование
    Redux store:
        все урны в пределах выбранной области

    Модель пользователя системы:
        личный кабинет
            водитель
                регистрация водителя
                карта маршрута
                сигналы о заполненности урн
            оператор
                назначение водителям маршрутов
                сигналы о заполненности урн
                добавление/удаление урн
        Redux store:
            текущий пользователь системы со всеми вытекаущими данными



2. Сервер
Node js приложение для работы с облачным сервисом Microsoft Azure
    Сбор статистики по загруженности мусорок (Обработка данных, сбор данных в базу данных) 
    Обучение модели (обучение мродели на основе данных из базы данных) 
    Прогнозирование загруженности на любое время вперед (Создание предсказательной модели, генерация данных прогнозов) 
    Отправка сообщений пользовтелям системы (Отправка оповещения для пользователей: ссылка на карту, с иаршрутом движения до урны с учетом пробок и рекомендованному времени очистки)
